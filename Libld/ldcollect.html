NAME

ldcollect - collects garbages in the log file of libld(3)

SYNOPSIS

ldcollect [file]

DESCRIPTION

The ldcollect program collects garbages in file, writing them to the standard output. The file should be the log file of libld(3). A format of the output is the same as that of the log file. If file is absent, ldcollect reads from the standard input.

SEE ALSO

ldbrowse(1), libld(3)
