Release Notes

リリースノート 20080908 (2008-09-08)

  • madangler-20080908.tar.bz2をリリース。
  • コンパイルの際に「Exception in thread "main" java.lang.OutOfMemoryError: Java heap space」でビルドが失敗することがあるバグを修正。
  • ビルド完了の通知メールに含まれるビルド結果のURLをブラウザによっては正しく開けないバグを修正。
  • History#fireCurrentHistoryState()を使用するようにした。

リリースノート 20080903 (2008-09-03)

  • madangler-20080903.tar.bz2をリリース。
  • libcgi 20080903に対応。

リリースノート 20080830 (2008-08-30)

  • madangler-20080830.tar.bz2をリリース。
  • GWT 1.5に対応。
  • ジェネリクスを使用するようした。
  • ダイアログの表示にアニメーションを使用するようにした。

リリースノート 20080608 (2008-06-08)

  • madangler-20080608.tar.bz2をリリース。
  • ダイアログ表示中は画面を薄暗くするオーバレイを表示するようにした。
  • 新規プロジェクトを生成するときに設定を既存のプロジェクトからコピーした場合、もう一度プロジェクトを新規作成しようとすると、設定のラジオボタンが両方チェックされるバグを修正。
  • Linuxで親プロセスでマスターデバイスを読み始める前にスレーブデバイスをクローズするようにして、子プロセスの終了の検出にシグナルを使わないようにした。

リリースノート 20080525 (2008-05-25)

  • madangler-20080525.tar.bz2をリリース。
  • Ubuntu 7.10とopenSUSE 10.3でビルドできるようにした。
  • iconvのパスをビルドの際に変更できるようにした。
  • getPropertyリクエストのレスポンスの文字列がヌルで終端していないバグを修正。

リリースノート 20080510 (2008-05-10)

  • madangler-20080510.tar.bz2をリリース。
