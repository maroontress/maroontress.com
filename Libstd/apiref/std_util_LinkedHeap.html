リンクによるヒープです。

ヒープは生成するときに指定された比較関数で要素を順序付けます。ヒープのルートの要素は常に最小の要素になります。

ヒープは要素となるオブジェクトについて一切関与しません。

LinkedHeap_create

union LinkedHeap LinkedHeap_create( int (*compare)(void *, void *))
ヒープを生成します。

compareには第1引数と第2引数の要素の大小を比較する関数を指定します。第1引数と第2引数が等しい場合は0、第1引数が小さい場合は負の整数、第2引数が小さい場合は正の整数を返す必要があります。

戻り値が参照するオブジェクトはLinkedHeap_free()で解放しなければなりません。

生成に失敗した場合はstd_lang_Object_NULLを返します。

Parameters:
compare - 要素を比較する関数のポインタ
Returns:
ヒープのインスタンス、またはstd_lang_Object_NULL

LinkedHeap_forEach

void LinkedHeap_forEach( union LinkedHeap this,
void (*callback)(void *))
ヒープに含まれる要素を引数として、指定されたコールバック関数を呼び出します。

ヒープに含まれるそれぞれの要素を引数としてcallbackを呼び出します。callbackを呼び出す回数はヒープに含まれる要素の個数と同じになります。すべての要素が1回ずつ呼び出されることは保証されますが、要素と呼び出しの順序は不定です。

callbackからヒープにアクセスすることはできません。

Parameters:
this - instance.
callback - 要素を引数として呼び出されるコールバック関数
